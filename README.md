## Operating Model

<p align="center">
  <img src="./assets/profile-command-flow.svg" alt="Mobile command to 24/7 finance automation command center" width="100%">
</p>

For approved environments, mobile instructions can route through Telegram or approved mobile AI access to Hermes, OpenClaw, Codex, or Claude-style assistants. Dedicated Mac mini worker nodes keep the workspace online, the finance engines run the work, validation checks the outputs, and the final package becomes a CEO/CFO-ready summary with findings, exceptions, links, and an email draft.

For conservative environments, the same control pattern runs without Hermes, OpenClaw, Telegram, or autonomous agents. The enterprise-safe demo uses Python, Excel-compatible workbooks, Markdown/JSON evidence, tests, CI, and human-gated review.
